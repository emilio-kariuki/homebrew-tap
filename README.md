# homebrew-tap

Homebrew tap for [ngstone](https://github.com/emilio-kariuki/ngstone), a self-hosted HTTP tunnel and personal ngrok alternative.

## Usage

```sh
brew tap emilio-kariuki/tap
brew install emilio-kariuki/tap/ngstone
```

Or in one step:

```sh
brew install emilio-kariuki/tap/ngstone
```

This installs both `ngstone` (the agent CLI) and `ngstoned` (the edge server).
